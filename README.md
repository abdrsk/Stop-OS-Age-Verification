## Maintainence
> by abdrsk
  
If FOSS OS projects (like many BSD's and Linux distributions) fail to comply with
the law, they might face bills more expensive than their budget, making the
developers have to work elsewhere for money or possibly take down the project,
causing maintainance latencies and security exploits to accur.

These projects are used in various places like most websites (including [usa.gov](https://www.usa.gov/)) and
embedded devices. Without them, the internet basically wouldn't exist. Most
security maintainence are done by people using a Linux distribution on their
devolopment workflow.

If FOSS OS' are taken down, it will cause a chain reaction of:  
No development workflow -> no developers -> no maintainence/maintainers -> security flaws. as consequence, most of the internet breaks apart into a state from where it would not recover from.
